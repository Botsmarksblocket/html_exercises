Git

git add <Filnamn> (Lägg till filnamn i stagning)
git commin -m "meddelande som syns i git log" Lägger till alla stagade filer (från git add) i senaste changeset
git branch <branchnamn> skapa branch med namn <branchnamn>
git checkout <branchnamn> byt branch till <branchnamn>

Terminal

ls (lista filer i mapp)
dir (samma som ls men funkar inte på Mac)
mkdir <mappnamn> (skapa ny mapp)
cd <mappnamn> (navigera till mappnamn, tänk att det är samma som att dubbelklicka i utforskaren)
cd .. (navigera till "föräldrar"/föregående mapp)
cd ~ (navigera till "förälder"/hem mapp)
touch <filnamn> (skapa tom fil med namn <filnamn>)
Notepad.exe <filnamn> (öppna/editera filen <filnamn> på Windows)
